Простое консольное приложение на Python для обработки текста и переворота слов.

Входные данные: текст. 
Результат работы алгоритма: массив слов входного текста, каждое из которых развернуто.

Функциональность

  - Ввод исходного текста (вручную или случайным образом)
  - Выполнение алгоритма переворота слов
  - Вывод результата обработки
  - Завершение работы программы
    
Разработка выполнена в стиле нисходящего проектирования:

Итерация 1: структура программы, все функции как заглушки.
Итерация 2: полная логика работы меню и зависимостей.
Aлгоритм оставлен как заглушка

После нажатия кнопки History откроется страница с историей коммитов, 
Первый коммит - итерация 1 (практическая работа №8), Второй коммит - итерация 2 (практическая работа №10).
