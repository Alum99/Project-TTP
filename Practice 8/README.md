Простое консольное приложение на Python для обработки текста и переворота слов.

Входные данные: текст. 
Результат работы алгоритма: массив слов входного текста, каждое из которых развернуто.

Функциональность

  - Ввод исходного текста (вручную или случайным образом)
  - Выполнение алгоритма переворота слов
  - Вывод результата обработки
  - Завершение работы программы
